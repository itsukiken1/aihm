# AIHM Tools

Free tools for cross-border e-commerce sellers sourcing from China.

## Tools

| Tool | Type | Description | Link |
|------|------|-------------|------|
| **SourceCalc** | Chrome Extension | Calculate profit on 1688.com product pages | [GitHub](https://github.com/itsukiken1/sourcecalc) |
| **BulkCalc** | Web App | Batch profit calculator for multiple products | [Use it](https://itsukiken1.github.io/bulkcalc/) |
| **ShipCalc** | Web App | Shipping cost estimator from China | [Use it](https://itsukiken1.github.io/shipcalc/) |

## Features

- **100% free** — no hidden costs, no premium tier (yet)
- **No signup** — just open and use
- **No data collection** — all calculations run in your browser
- **Open source** — all code is public

## Who is this for?

- Amazon FBA sellers sourcing from 1688/Alibaba
- Shopify store owners importing from China
- Dropshippers evaluating product margins
- Anyone importing goods from China
